Credit Card Customer & Transactions Data Analysis

1.	Objective 

-	Build a weekly credit card customers and transactions dashboard that provides insights into performance metrics and trends enabling stakeholders to monitor and analyze credit card operations effectively.


2.	Goal

•	Measure Total Revenue, Total Interest earned, Total Transactions, Customer Satisfaction Score.
•	Week-over-Week Revenue Growth 
•	Customer Segmentation & Card Category Performance
•	Quarterly Revenue & Transactions growth
•	Identify spending and card usage patterns.  


3.	Analysis done
1)	Import data from SQL database into Power BI
-	Connected Power BI Desktop with SQL Server and pulled the required tables into Power BI.

2)	Data Processing
-	Created aggregated columns Age Group and Income Group using DAX functions for customer segmentation to understand spending patterns across age and income groups.
-	Created DAX measures to see week-over-week growth in revenue 


4.	Results
Built an interactive Power BI dashboard showing
-	KPIs measuring total revenue, total interest earned, total transaction volume, Customer Satisfaction score.
-	Chart visual showing Week-over-Week revenue growth trend.
-	Table showing customer segments and card category performance.
-	Chart visuals showing patterns in customer segments.
-	Charts visual to identify quarterly revenue and transactions volume growth.
-	Visuals showing spending and card usage patterns.
